# CitibankHK PWA

A Progressive Web App (PWA) that replicates the Citibank HK mobile app UI for iOS, based on the provided screenshots and screen recording.

## Features
- Dashboard matching the first screenshot
- Click "Citi利息Booster" → Account detail page (exactly like the video)
- "顯示所有交易" → Full transaction listing with search
- Fully installable on iOS Home Screen

## Setup
1. Add your `icon-192.png` and `icon-512.png` to this folder
2. Push to GitHub → Enable GitHub Pages (main branch)
3. Open in Safari → Share → "Add to Home Screen"

See `index.html` for the full interactive demo.
